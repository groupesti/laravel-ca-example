# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by email to: dev@groupesti.com

You will receive a response within 72 hours.

Include: affected version, description, reproduction steps, potential impact.

## Disclosure Policy

We follow a 90-day coordinated disclosure policy.

## Important Note

This package provides **demo and example** data for development and testing purposes. It should **never** be used in production environments with real certificate authority operations. Demo certificates and keys generated by this package are not suitable for production use.
