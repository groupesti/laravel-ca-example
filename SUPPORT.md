# Support

## Getting Help

- **Bug reports**: [GitHub Issues](https://github.com/groupesti/laravel-ca-example/issues) (bugs only)
- **Questions and discussions**: [GitHub Discussions](https://github.com/groupesti/laravel-ca-example/discussions)
- **Documentation**: [README](README.md)

## Before Opening an Issue

Please include the following information in any bug report:

- Laravel CA Example package version
- PHP version (must be 8.4+)
- Laravel version (12.x)
- Steps to reproduce the issue
- Expected vs actual behavior

## Important

GitHub Issues are reserved for **bug reports and feature requests**. For general support questions, please use GitHub Discussions.
